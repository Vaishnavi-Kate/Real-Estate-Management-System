# Real Estate Web Application

## Project Overview
Real Estate Web Application is a robust, full-stack Java-based web application engineered to simplify modern real estate transactions. The platform bridges the gap between property seekers and providers by offering a centralized hub to seamlessly search, buy, rent, and list diverse residential accommodations. Designed to cater to varied urban housing requirements, the system supports standard property transactions along with specialized categories like Paying Guest (PG) accommodations, student hostels, and co-living flatmate searches.

---

## Key Features
- User Authentication & Security: Secure registration and login modules for distinct user roles (Buyers, Sellers, Tenants, and Admins).
- Dynamic Search & Filtering: Advanced search functionality enabling users to filter properties by location, budget, layout, and accommodation type.
- Comprehensive Listings: Dedicated modules to browse, compare, and finalize purchases or rental agreements for flats and independent houses.
- Co-Living & Alternative Housing: Specialized discovery portals for tracking available PG accommodations, hostels, and finding compatible flatmates.
- Property Management: Intuitive dashboard for owners to add detailed property specifications, manage pricing, and execute full CRUD operations.
- Multimedia Integration: Capability for property owners to upload and showcase high-resolution images of their real estate listings.
- Centralized Admin Control: Powerful admin management system to monitor user activity, verify listings, and ensure platform safety.

---

## Technologies Used
- Backend Core: Java, Java Server Pages (JSP), and Servlets
- Database & Connectivity: MySQL (Relational Database Management System) and JDBC API
- Frontend Design: HTML5, CSS3, JavaScript, and Bootstrap Framework
- Server Environment: Apache Tomcat Web Server
- Development Tools: Eclipse IDE / VS Code and Git Version Control

---

## Project Outcome
Developed a complete full-stack web application with database integration and CRUD operations for property management.
